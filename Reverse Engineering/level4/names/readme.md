# 🧩 Reverse Engineering Challenge: names

## 📖 문제 개요
- **문제 이름**: names 
- **출처**: 드림핵  
- **키워드**: 커스텀 해쉬,해시 충돌, 브루트포스  
- **설명**: 커스텀 해쉬 연산을 한 결과를 보고 특정 값을 입력해야지 플래그를 얻을 수 있는 문제  

---

## 🛠️ 사용 도구
- **IDA**: 정적 분석  
- **GDB**: 동적 분석  
- **PyCharm**: 코드 작성  

---

## 🚀 풀이 방법
- 오직 해쉬 연산만 보고 브루트포스를 통해 입력해야 하는 값을 찾기

---

## 📁 파일 구성
- **main** : 키-이름으로 된 txt를 불러와 해쉬 연산을 한 뒤 검색 기능 제공, 종료되면 그 해쉬 연산 결과 - 이름을 dump.txt에 저장
- **dump.txt** : 해쉬 결과 - 이름 저장 txt

## 🧠 접근 방법
- 시작하기에 앞서 0796 DH{**REDACTED**}라는걸 dump.txt를 볼 수 있다. 즉 숫자를 입력하면 0796인걸 찾아야한다.
- 즉 이름은 해쉬에 영향을 안주기에 해쉬 연산을 1~0xFFFF까지 돌리면 된다. 즉 브루트포스다.
