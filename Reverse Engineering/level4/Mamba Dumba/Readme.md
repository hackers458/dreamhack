# 🧩 Reverse Engineering Challenge: Mamba Dumba

## 📖 문제 개요
- **문제 이름**: Mamba Dumba  
- **출처**: 드림핵  
- **키워드**: pyc 파일 구조, 디컴파일, 내장 모듈 동적 불러오기(?)
- **설명**: 정렬을 통한 배열을 sha256으로 암호화한 결과를 플래그로 출력  

---

## 🛠️ 사용 도구
- **PyCharm**: 코드 작성  
- **Pycdc** : pyc 디컴파일러
---

## 🚀 풀이 방법
- 직렬화된 바이너리의 작동 방식을 분석 한 뒤 요구하는 플래그를 찾기

---

## 📁 파일 구성
- **chall.py** : 답을 입력하면 플래그를 입력하는 프로그램

## 🧠 접근 방법
해당 verify 코드를 보면 %256으로 비트 손실이 일어날수도 있다.
그러므로 verify 코드 중 입력 가능한 문자로 브루트포스를 하여 각 연산의 결과와 맞게 구하면 된다.

---


## 📚 공부한 내용
- 이거 pyc의 헤더구조를 아느냐 문제인줄 알았으나 marshal 직렬화된 코드 객체 (E3 ...부터 시작) 이걸 알아야 하는 문제였다. 하지만 디컴파일하는 방법이 익숙하니 이걸 더 선호한다.
