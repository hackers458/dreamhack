# 🧩 Reverse Engineering Challenge: hash-brown

📖 문제 개요

- 문제 이름: hash-brown
- 출처: 드림핵
- 키워드: MD5,브루트포스
- 설명: MD5의 원리와 취약점을 이해하면 된다.

# 🛠️ 사용 도구

- IDA: 정적 분석
- pycharm : 레인보우 테이블 작성

# 🚀 풀이 방법

- MD5의 원리와 취약점을 이해한 뒤 레인보우 테이블을 만들어 직접 값을 찾아 플래그를 만들면 된다.
# 📁 파일 구성

- hash-brown : 입력한 글자와 저장된 메모리값과 비교하여 맞으면 플래그를 출력하는 프로그램(입력값 그대로 플래그를 출력하니 사실 플래그를 입력해야하는 샘)

# 🧠 접근 방법
메모리에 저장된 값들을 해쉬값으로 만들어 (리틀엔디언 -> 빅엔디언) 레인보우 테이블로 저장된 텍스트 파일로 찾아 해당하는 3글자와 일치하는 값들을 찾으면 된다.
