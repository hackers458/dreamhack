# 🧩 Reverse Engineering Challenge: Snaky

## 📖 문제 개요
- **문제 이름**: Snaky  
- **출처**: 드림핵  
- **키워드**: DFS(BFS),3차원 미로,배열 접근
- **설명**: 정렬을 통한 배열을 sha256으로 암호화한 결과를 플래그로 출력  

---

## 🛠️ 사용 도구
- **IDA**: 정적 분석  
- **PyCharm**: 코드 작성  

---

## 🚀 풀이 방법
- 시작점과 도착점이 주어진 상태에서 DFS(BFS)로 최단경로(또는 가능한 경로)를 구하면 된다.

---

## 📁 파일 구성
- **chall** : 플래그를 입력하면 그 플래그의 값에 따라 XYZ방향으로 이동하는 프로그램, 맞으면 입력이 곧 플래그

## 🧠 접근 방법
- 시작점은 xyz를 두었을때 각각 12,12,24이며 도착점은 12,12,0이다. 이를 나는 bfs로 하여 모든 경로를 탐색하여 도착점을 갈 수 있게 코드를 짜면 된다. 또한 해당 좌표로 이동할때 마다 간 방향의 알파벳도 추가해주면 된다.


---

## 📚 공부한 내용
- 배열의 접근법을 완전히 이해했다. byte_404060[625 * x + 25 * y + z]를 보았을때 본능적(?)으로 3차원 배열처럼 생겼으나 최대 인덱스를 어떻게 알까
- 생각을 했는데 2차원 배열을 메모리로 접근할때 [x][y]라고 하면 y의 크기 + x의 값을 해야지 접근할 수 있는 것 처럼 저것도 y앞에 25는 x의 크기, x앞에 있는 625는 25를 나눈 값인 25, 즉 y의 크기, 그리고 저 바이트의 전체 범위(크기)로 나눠주면
- 되는 걸 알 수 있었다.
