# 🧩 Reverse Engineering Challenge: carta

## 📖 문제 개요
- **문제 이름**: carta
  
- **출처**: 드림핵  
- **키워드**: 함수 분석  
- **설명**: 카드 게임의 규칙을 분석하고 모든 카드를 다 짝을 지으면 이기는 게임

---

## 🛠️ 사용 도구
- **IDA**: 정적 분석  
- **GDB**: 동적 분석  
- **PyCharm**: 코드 작성  

---

## 🚀 풀이 방법
- 카드 게임의 규칙을 이해한 뒤 게임을 이기면 플래그를 출력하는 프로그램

---

## 📁 파일 구성
- **chall** : 카드게임 실행 프로그램, 이기면 플래그를 출력
